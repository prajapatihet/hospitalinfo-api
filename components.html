API Information

Available Endpoints

Component Path Method Description
Accordions /accordions/accordions.json GET Retrieve information about Accordions components.
Backgrounds /backgrounds/backgrounds.json GET Retrieve information about Backgrounds components.
Breadcrumbs /breadcrumbs/breadcrumbs.json GET Retrieve information about Breadcrumbs components.

How to Use the API

1. Retrieve Information About Accordions Components

To retrieve information about Accordions components, send a GET request to:

Endpoint: /components/accordions/accordions.json

Example Request:

GET https://components-api.hetprajapati80.workers.dev/components/accordions/accordions.json

2. Retrieve Information About Backgrounds Components

To retrieve information about Backgrounds components, send a GET request to:

Endpoint: /components/backgrounds/backgrounds.json

Example Request:

GET https://components-api.hetprajapati80.workers.dev/components/backgrounds/backgrounds.json

3. Retrieve Information About Breadcrumbs Components

To retrieve information about Breadcrumbs components, send a GET request to:

Endpoint: /components/breadcrumbs/breadcrumbs.json

Example Request:

GET https://components-api.hetprajapati80.workers.dev/components/breadcrumbs/breadcrumbs.json
