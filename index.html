API Information

This API provides the following endpoints:

Path Method Description Example
/bloodbank GET Retrieve all blood banks information. /bloodbank
/bloodbank/{id} GET Retrieve specific blood bank information by ID. /bloodbank/127
/bloodbank?name={name}&loc={loc}&add={add} GET Filter blood banks by name, location, and address. /bloodbank?name=Kailash&loc=Vadodara&add=Goraj
/donor GET Retrieve all donor information. /donor
/donor/{id} GET Retrieve specific donor information by ID. /donor/51
/donor?location={location}&bloodgroup={bloodgroup} GET Filter donors by location and blood group. /donor?location=Vadodara&bloodgroup=A+
