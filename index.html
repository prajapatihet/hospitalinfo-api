API Information

Available Endpoints

Path Method Description
/bloodbank GET Retrieve all blood banks information.
/bloodbank/{id} GET Retrieve specific blood bank information by ID.
/bloodbank?name={name}&loc={loc}&add={add} GET Filter blood banks by name, location, and address.
/donor GET Retrieve all donor information.
/donor/{id} GET Retrieve specific donor information by ID.
/donor?location={location}&bloodgroup={bloodgroup} GET Filter donors by location and blood group.

How to Use the API

1. Retrieve All Blood Banks Information

To retrieve all blood banks information, you need to send a GET request to the following endpoint:

Endpoint: /bloodbank

Example Request: https://info-api.hetprajapati80.workers.dev/bloodbank

2. Retrieve Specific Blood Bank Information by ID

To retrieve information for a specific blood bank by ID, use this endpoint:

Endpoint: /bloodbank/{id}

Example Request: https://info-api.hetprajapati80.workers.dev/bloodbank/127

3. Filter Blood Banks by Name, Location, and Address

To filter blood banks based on name, location, and address, use the following endpoint with query parameters:

Endpoint: /bloodbank?name={name}&loc={loc}&add={add}

Example Request: https://info-api.hetprajapati80.workers.dev/bloodbank?name=Kailash&loc=Vadodara&add=Goraj

4. Retrieve All Donor Information

To get a list of all donors, send a GET request to:

Endpoint: /donor

Example Request: https://info-api.hetprajapati80.workers.dev/donor

5. Retrieve Specific Donor Information by ID

To access information for a specific donor by ID, use this endpoint:

Endpoint: /donor/{id}

Example Request: https://info-api.hetprajapati80.workers.dev/donor/51

6. Filter Donors by Location and Blood Group

To filter donors based on location and blood group, use this endpoint:

Endpoint: /donor?location={location}&bloodgroup={bloodgroup}

Example Request: https://info-api.hetprajapati80.workers.dev/donor?location=Vadodara&bloodgroup=A+
